How do I get my unit list?

Moogle Fuser makes use of ffbeequip's Sync utility. You can read an overview here and here.
  1. Link your FFBE account to Google or Facebook. (Please, do it even if you're not using Moogle Fuser).
  2. Install the FFBE Sync plugin for Chrome (for Chrome in PC or Opera) or Firefox (for PC and Android).
  3. Open the plugin and login with the Google/Facebook account that you use for FFBE.
  4. Wait until yor data is ready.
  5. In the Download section, click "Units". Save the .json file in a directory.

Using Moogle Fuser

Now the easy part
  1. Select the language that you use in FFBE. Available options are German, English, Spanish, French, Korean and Chinese (zh).
  2. Select the file containing your unit list created by ffbeequip Sync.
  3. Click "Show moogles!". In a couple of seconds, the list of your moogles should be ready!
  4. The list is ordered by the TMR's alphabetic order. In FFBBE go to Units > Merge. Select Sort > TMR name. The units should be listed in the same order, so you can follow easily the list.
